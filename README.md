# QuickShifter with arduino
 ## Requirements 
 -Arduino Nano
 
 -Wires
 
 -2 5v relays with NO, NC (1 relay for single cylinder 2 relays for 2,4 cylinders)
 
 -Reed Switch 
 
 -Neodymium magnet
 
 -Plastic box
 
 -Lm7805 Voltage Regulator
